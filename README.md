# 🤖 Artificial Intelligence Lab Manual

This repository contains a collection of practical experiments performed as part of the Artificial Intelligence laboratory coursework. The experiments are designed to provide hands-on experience with core concepts of Machine Learning, Deep Learning, Natural Language Processing, and Recommendation Systems.

The implementations focus on clarity, understanding, and academic learning, making this repository useful for both study and reference purposes.

---

## 📌 Overview

The lab manual covers a wide range of AI topics including:

- Machine Learning fundamentals  
- Supervised learning algorithms  
- Deep learning architectures  
- Natural Language Processing techniques  
- Information extraction and summarization  
- Recommendation systems  
- Reinforcement learning  

Each experiment includes problem understanding, implementation, and observations.

---

## 📘 Introduction to Machine Learning

This experiment introduces the basic concepts of Machine Learning such as supervised and unsupervised learning, training vs testing data, and real-world applications. It builds the foundation required for understanding advanced algorithms.

---

## 📈 Linear Regression

This experiment focuses on implementing Linear Regression to model relationships between dependent and independent variables. It includes:

- Understanding regression concepts  
- Model training and prediction  
- Evaluation using metrics like Mean Squared Error  

---

## 📊 K-Nearest Neighbors (KNN)

KNN is implemented as a simple yet powerful algorithm for classification and regression. This experiment covers:

- Distance metrics (Euclidean, Manhattan)  
- Choosing optimal K value  
- Model accuracy and performance evaluation  

---

## 🧠 Convolutional Neural Networks (CNN)

This experiment introduces deep learning using CNNs, primarily for image data. Key concepts include:

- Convolution operation  
- Feature extraction  
- Pooling layers  
- Model architecture design  

---

## 🗣️ Natural Language Processing (NLP)

This experiment focuses on text data processing techniques. It includes:

- Tokenization  
- Stopword removal  
- Stemming and Lemmatization  
- Text cleaning and preprocessing  

---

## 🔤 N-gram Model

The N-gram model is implemented for language modeling. It demonstrates:

- Probability of word sequences  
- Bigram and Trigram models  
- Text prediction concepts  

---

## 🔎 Word Embedding

This experiment explores vector representations of words, enabling machines to understand semantic meaning. Topics include:

- Word2Vec / embedding concepts  
- Similarity between words  
- Context-based representations  

---

## 🧾 TextRank for Keyword Extraction

Implements the TextRank algorithm to extract important keywords from a document using graph-based ranking techniques.

---

## 📄 TextRank for Document Summarization

This experiment applies TextRank to generate summaries by selecting the most relevant sentences from a document.

---

## 🎯 Content-Based Recommendation System

Builds a recommendation system based on item features. It includes:

- Feature extraction  
- Similarity calculation  
- Personalized recommendations  

---

## 🤝 Collaborative Filtering Recommendation System

This experiment focuses on recommending items based on user interactions. It covers:

- User-user similarity  
- Item-item similarity  
- Recommendation generation  

---

## 🎮 Reinforcement Learning

Introduces reinforcement learning concepts where an agent learns through interaction with the environment. Key topics include:

- Rewards and penalties  
- Policies and actions  
- Learning strategies
